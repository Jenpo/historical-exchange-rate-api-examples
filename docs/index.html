Historical exchange rate API and CSV examples

Copyable examples for fetching historical exchange rates as JSON or CSV, with links to FXpeek pages built for spreadsheets, reports, dashboards, and AI citation workflows.

API and CSV docs Endpoint examples for rates, history, and CSV exports. API / CSV answer hub Short answers and FAQ schema for AI citation. Spreadsheet answer hub Excel, Google Sheets, CSV import, and reporting workflows. GitHub repository JavaScript, Python, and shell examples.

Historical JSON

curl 'https://fxpeek.com/api/history?from=CNY&to=TRY&days=365'

CSV export

curl -L 'https://fxpeek.com/api/csv?from=CNY&to=TRY&days=365' -o cny-try-history.csv

When to use these examples

  • Use JSON for app, dashboard, and script integrations.
  • Use CSV for Excel, Google Sheets, pandas, and monthly report files.
  • Use pair pages or the annual reference report when a human-readable citation is needed.
  • FXpeek rates are reference data for lookup and reporting, not transaction quotes.
